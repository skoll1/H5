新的表单元素

  • datalist:输入提示作用
  • keygen:提供一种验证用户的可靠方法。用于生成表单的密匙对生成器字段。当提交表单的时候，一个私匙存储在客户端，公匙则被发送到服务器
  • output:标签用于定义不同类型的输出，比如脚本的输出。

表单的属性

  • form
    • autocomplete:自动完成
    • novalidate:规定在提交表单的时候不应该验证form或input域。
  • input
    • autofocus:自动获得焦点
    • from:位于表单外面的input规定自己属于哪个表单
    • formaction:描述表单提交的url地址，会覆盖form元素中的action属性-用于type='submit',type='image'
    • height,width:
    • min,max,step-date,picker,number,range类型的input
    • multiple:input元素可以选择多个值-email,file
    • pattern:属性描述了一个正则表达式用于验证input元素的值
    • require:必须在提交之前填写输入域
